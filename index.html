Text Formatting

there are three main types of text formatting. those are bold italics and underline.

inline vs block elements

a block element moves the content to a new line. the paragraph and header elements are block elements. an inline elementdoes not display its content on a new line. it keeps the content on the same line as the previous content. bold, italics and underline are all inline elements

images

an image elementis an inlineelement which inserts pictures into the the document. an image element is self-closing, which means it does not have a closing tag. it has one major attribute, src, which is a filepath that points to the images' location.those filepaths can either be absolute or relative

simple image example

image attributes

there are 4 attributes commonly used for images. width and height are used to resize the images. alt allows to use text description of images for accessibility and broken images. title allows a hover-over text to appear.

width and height

the width and height can be measured in many different ways, but the more common ones are pixels and percentages. using pixels,you set the specific, precise size based on pixels.using percentages, you set the dimensions based on a percentage of the entire page.in either case, setting the width alone respects aspect ratio, while setting both does not.

alt attribute

EVERY picture should at LEAST have the alt attribute. the alt attribute helps with screen readers for visually impaired and in the cases that the image is broken.
